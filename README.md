# File server

This file server is used to upload and download files using `REST APIs`

  - `POST` - upload a file
  - `GET` - to retrieve a file

### Requirements
- `Python: 3.7.5`
- `Django: 3.0.2`
- `Postgres 11.5`
- `Memcached 1.5.10`
- `gettext package`


### Installation
- `pipenv install`
- `pipenv shell`
- `python manage.py runserver`
